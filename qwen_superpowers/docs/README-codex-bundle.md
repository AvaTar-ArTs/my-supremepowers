# Superpowers for Codex

A minimal, production-ready bundle that makes superpowers the default workflow in Codex.

## What This Adds

- A bootstrap flow that auto-loads `using-superpowers`
- A skill discovery command (`superpowers-codex`)
- Clear default instructions for Codex and agent usage

## Included

- `bin/superpowers-codex` CLI wrapper
- `docs/superpowers-bootstrap.md`
- `docs/INSTALL.md`
- `templates/AGENTS.md` snippet for default activation

## Quick Start

1. Copy this folder where you want to host the bundle.
2. Follow `docs/INSTALL.md`.

## Default Behavior

Once installed, Codex will bootstrap superpowers at session start and auto-load `using-superpowers`.
