# Qwen Use Cases & Triggers - Advanced Capabilities

**Purpose:** Define use cases for the Qwen ecosystem, how to trigger them, and what to link to so the agent can **recall** the right flow or skill within the Qwen context.

**Recall:** When the user's goal or phrasing matches a use case below, **link** to the listed flow/skill/doc and **run** the appropriate Qwen capability as specified.

---

## 1. Creative / Feature Work (idea → done) — **Qwen Superpowers**

| | |
|--|--|
| **Use case** | User has an idea, feature, or multi-step change; wants to go from idea to done and reviewed. |
| **Trigger** | "I have an idea," "build a feature," "add X," "implement Y," "run superpowers," "plan before code," "brainstorm then implement," or any non-trivial creative/implementation ask. |
| **Link / recall** | **Qwen Workflow Bootstrap**; Qwen Logic Flows. **Qwen has superpowers to run.** |
| **Run** | **Qwen Brainstorming** → design doc → **Qwen Writing Plans** → plan doc → **Qwen Executing Plans** → **Qwen Verification Before Completion**. |
